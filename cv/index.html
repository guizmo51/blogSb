Projet BetterCallUber

BetterCallUber est le résultat de ma participation à un hackathon organisé par le service de VTC Uber. Ce projet est une application web permettant de réserver instantanément ou de manière programmée un VTC. D’autres fonctionnalités sont en cours de développement comme le déclenchemnt d’une course à l’aterrisage d’un avion ou train.

Ce projet sera prochainement détaillé sur ce blog.

Technologies : PHP5, Symfony 2.6, AngularJS, Bootstrap, consommation & réalisation d’API, HTTPS/SSL, JsonWebToken,

Projet RunRunRun

Il s’agit d’une application pour Firefox OS, le nouvel OS mobile de Mozilla. RunRunRun vous permet d’enregistrer votre parcours lorsque vous sortez faire un footing ou du vélo. Les technologies employées sont HTML5, JavaScript et CSS. Cette application utilise IndexedDB et la géolocalisation avec les API HTML5.

Voir le dépôt GitHub de ce projet

Projet Boostadopt

BoostAdopt est un projet permettant d’améliorer sa visibilité sur un site de rencontre français. Il permet aux utilisateurs de visiter automatiquement un grand nombre de profils définis grâce à un moteur de recherche.

Suite à une mise en demeure ce projet est désactivé.

Technologies : PHP, CakePHP, AngularJS, Bootstrap, Javascript natif pour extension Chrome, crawlers, APIs

Projet MapBoostadopt

Le projet BoostAdopt a permis d’agréger plusieurs dizaines de milliers de profils d’utilisateurs européens d’un site de rencontre.

Cette application affichait tous ces profils sur un système cartographique. Ce projet a été désactivé suite à une mise en demeure.

Technologies : Bootstrap, NodeJS, ExpressJS, MongoDB, MapQuest (OpenStreetMap, Google Maps), Leaflet, API de données géographiques.
