Projet Boostadopt

BoostAdopt est un projet permettant d’améliorer sa visibilité sur un site de rencontre français. Il permet aux utilisateurs de visiter automatiquement un grand nombre de profils définis grâce à un moteur de recherche. Suite à une mise en demeure ce projet est désactivé. Technologies : PHP, CakePHP, AngularJS, Bootstrap, Javascript natif pour extension Chrome, crawlers, APIs
